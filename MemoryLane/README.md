# MemoryLane

## Introduction

MemoryLane is a digital journaling application that helps users preserve and revisit their cherished memories. The app allows users to create, organize, and reflect on personal moments through text entries, photos, and tags. It solves the problem of forgotten moments by providing an elegant, searchable memory repository with a beautiful timeline interface.

## Features

- **Timeline View**: Chronological display of all memories
- **Memory Creation**: Add memories with titles, descriptions, dates, locations, and tags
- **Album Organization**: Group related memories into albums
- **Tagging System**: Categorize memories with custom tags
- **Responsive Design**: Works on all device sizes
- **Authentication**: Secure user signup and login
- **Dark Mode**: Eye-friendly dark theme option
- **Random Memory**: "Reminisce" feature shows random past memories

## Design Decisions & Assumptions

- Chose Firebase for backend services (Authentication + Firestore) for rapid development
- Implemented a clean, minimalist UI focused on content
- Designed for emotional resonance with warm color palette
- Assumed users would want both chronological and categorical organization
- Used a mobile-first responsive design approach
- Implemented client-side routing for a single-page application feel
